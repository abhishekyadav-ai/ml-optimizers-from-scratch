# ML Optimizers From Scratch

This repository contains implementations of common optimization algorithms used in machine learning and deep learning.

The goal is to understand how models learn by implementing these algorithms from scratch using Python,NumPy,Pandas and matplotlib.

## Implemented Optimizers

- Gradient Descent

## Why This Project

Libraries such as PyTorch and TensorFlow implement these optimizers internally.  
This project focuses on understanding the mathematical ideas behind them by coding them manually.

## Structure

gradient_descent → batch gradient descent implementation  
stochastic_gradient_descent → SGD implementation  
momentum → momentum-based optimization  
rmsprop → RMSProp optimizer  
adam → Adam optimizer  

More implementations will be added gradually.
